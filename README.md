# SafeTiles

SafeTiles is a  web puzzle game look likes a minesweeper game...
The goal is simple: reveal every safe tile on the board while avoiding hidden bombs. 
Each revealed tile displays the number of bombs located in the surrounding cells, helps you make safer decisions as you progress.

The game is easy to learn and play. You can restart at any time....

## Features

* 6×6 game board which is pretty cool
* Random bomb placement every round ( you never know which one will kill you )
* Nearby bomb number indicators
* Safe tile progress tracker
* Win and lose based on your perfomance
* Live game timer
* Restart button for quick happiness


## How to Play

1. Click any tile to reveal it.
2. If the tile contains a bomb.... the game ends.
3. Safe tiles display the number of bombs around them.
4. Use the revealed numbers to determine safer areas.
5. Reveal all non-bomb tiles to win the game.
6. Try to finish as quickly as possible to set a better time.
